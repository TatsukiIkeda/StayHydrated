## 概要

熱中症対策アプリです

水分を補給したら摂取量の記録をつけましょう。

どれだけ水分を撮ったかがグラフで確認できます。

【機能・特徴】

・水分摂取量の把握

・水分摂取を時間ごとに表示

・目標水分摂取量の設定


【使用方法】

１、現在地の天気を調べましょう

２、熱中症指数を参考に摂取する水分摂取量を目標設定しましょう！

３、水分を摂取したら飲んだ水分量を選択して記録するだけです。

４、熱中症にならないよう、こまめに水分摂取をしましょう！

【注意】

※天気予報APIがサービス終了により、最高気温と熱中症指数が現在できません
